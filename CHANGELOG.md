## Como Adicionar Entradas

### Tipos de Mudanças

- **Adicionado**: Para novas funcionalidades
- **Modificado**: Para mudanças em funcionalidades existentes
- **Depreciado**: Para funcionalidades que serão removidas
- **Removido**: Para funcionalidades removidas
- **Corrigido**: Para correções de bugs
- **Segurança**: Para vulnerabilidades corrigidas

### Exemplo

```markdown
## [1.1.0] - 2024-02-XX

### Adicionado
- Nova funcionalidade de exportação PDF
- Suporte a múltiplos idiomas

### Modificado
- Melhorada performance do dashboard
- Atualizado design do menu lateral

### Corrigido
- Corrigido bug no filtro de datas
- Corrigido problema de autenticação em alguns navegadores

### Segurança
- Atualizadas dependências com vulnerabilidades
- Implementada validação adicional de inputs
```
